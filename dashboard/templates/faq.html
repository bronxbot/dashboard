Support & FAQ

Find answers to common questions about BronxBot or get in touch with our support team for additional help.

Getting Started

How do I add BronxBot to my Discord server?

To add BronxBot to your Discord server, you need to be logged in with Discord OAuth. Once authenticated through our dashboard, you can manage the bot in servers where you have administrator permissions. The bot uses Discord's slash commands and requires appropriate permissions to function properly.

What permissions does BronxBot need?

BronxBot requires basic permissions including reading messages, sending messages, using slash commands, and managing roles for moderation features. The bot follows Discord's best practices and only requests permissions it actually needs to function properly.

How do I access the admin dashboard?

You can access the admin dashboard by logging in with your Discord account. The dashboard provides real-time statistics, server management, command execution metrics, and configuration options. You'll need administrator permissions in the servers you want to manage.

Commands & Features

What commands does BronxBot support?

BronxBot supports a variety of slash commands for moderation, utility, and fun. Common commands include server management, user moderation, statistics viewing, and interactive features. All commands are implemented as Discord slash commands for the best user experience.

How can I view command usage statistics?

The admin dashboard provides detailed command execution metrics including daily usage charts, total command counts, and performance trends. You can track which commands are most popular and monitor bot activity over time.

Can I customize BronxBot's settings for my server?

Yes! Through the admin dashboard, you can access server-specific settings, configure command permissions, set up moderation preferences, and customize the bot's behavior to match your server's needs. Each server has its own independent configuration.

Technical Support

BronxBot isn't responding to commands. What should I do?

First, check if the bot is online (green status). Ensure the bot has the necessary permissions in your server. Try using slash commands instead of prefix commands. If issues persist, check the dashboard for any error messages or contact our support team.

How can I report bugs or request features?

You can report bugs or request new features by joining our Discord support server. We actively monitor feedback and regularly update the bot with improvements and new features based on community suggestions.

Is my server data secure with BronxBot?

Yes, we take data security seriously. BronxBot only collects necessary operational data and follows Discord's data protection guidelines. We don't store personal messages or sensitive information, and all data transmission is encrypted.
